Active Directory &
Windows Server 2022

Déploiement d'un contrôleur de domaine sur Proxmox, gestion centralisée des utilisateurs, GPO et services réseau DNS/DHCP intégrés à l'Active Directory.

Windows Server 2022Active DirectoryGPODNSDHCPProxmox

📋 Contexte & objectifs

Déploiement d'une infrastructure Windows Server complète sur Proxmox, simulant l'infrastructure d'une PME avec domaine Active Directory centralisé. Ce projet est également une activité de l'épreuve E6.

Objectifs : contrôleur de domaine, gestion centralisée des comptes, politiques de sécurité GPO, services DNS et DHCP intégrés.

🛠️ Déroulement

  • 1

    Création VM Proxmox

    Provisionnement de la VM Windows Server 2022, paramétrage des ressources.

  • 2

    Installation AD DS

    Installation du rôle Active Directory Domain Services, promotion en contrôleur de domaine.

  • 3

    Structure OU & comptes

    Hiérarchie d'Unités d'Organisation par service, comptes utilisateurs et groupes de sécurité.

  • 4

    Configuration GPO

    Stratégies de groupe : restrictions d'accès, paramètres de sécurité, configuration du bureau.

  • 5

    DNS & DHCP

    Configuration des zones DNS intégrées, installation et configuration du rôle DHCP.

  • 6

    Tests & jonction domaine

    Jonction de postes Windows 10 au domaine, vérification de l'application des GPO.

⚠️ Difficultés rencontrées

  • Problème de résolution DNS lors de la jonction des postes clients — mauvaise configuration du DNS primaire.
  • GPO non appliquées immédiatement — nécessité de gpupdate /force et compréhension de l'héritage.
  • Conflits entre plages DHCP et adresses IP statiques sur le réseau Proxmox.

✅ Résultats obtenus

  • Contrôleur de domaine fonctionnel avec structure OU reflétant une organisation réelle.
  • GPO appliquées correctement sur les postes clients du domaine.
  • Services DNS et DHCP opérationnels et intégrés à l'Active Directory.

💡 Ce que j'ai appris

Ce projet m'a donné une vision concrète de l'administration Windows en entreprise, renforcée par la pratique chez V2SI. Les GPO permettent de gérer des dizaines de postes en quelques clics — compétence essentielle en SISR.
